### 😍 模块说明和安装

基础模块：

```bash
# 安装 cnpm
npm i -g cnpm

# 自动格式化代码
cnpm i --save-dev --save-exact prettier
cnpm i --save-dev @types/node 
```

开发安装：

```bash
# 字体
cnpm i misans

# UI 和图标
cnpm i antd@4.24.14
cnpm i @ant-design/icons
cnpm i moment

# 路由
cnpm i react-router
cnpm i react-router-dom

# 样式
cnpm i less less-loader

# 网络请求
cnpm i axios

# 数据
cnpm i valtio
```
